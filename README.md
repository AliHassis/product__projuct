 📊 Persistent Sales Management & Data Analytics System

A robust, object-oriented Python application designed for real-time sales tracking, inventory data persistence, and strategic business intelligence. Powered by *Pandas* and built using *OOP core principles*, this system automates retail data auditing, evaluates pricing constraints, and extracts high-level performance insights.

## 🚀 Key Features

- *Object-Oriented Data Modeling (OOP):* Utilizes structured entity classes to handle, validate, and parse user inputs safely.
- *Persistent Storage System:* Automatically updates and synchronizes real-time transactions into a flat-file database (products_project.csv) without wiping historical data.
- *Strategic Analytical Reporting:* Implements advanced Pandas grouping (groupby) to deliver instant categorical revenue breakdowns.
- *Automated Data Quality Audit:* Continuously monitors data health by tracking unique products, reporting missing fields, and identifying/removing redundant duplicate rows automatically.

## 📈 Strategic Insights Generated

The system analyzes retail performance metrics to expose critical management trends:
1. *Category Leaderboard:* Pinpoints the highest-grossing category using the idxmax() vector function.
2. *Data Cleansing Audits:* Evaluates missing inputs (isna()) and eliminates data duplication to maintain financial data integrity.

## 🛠️ Tech Stack

- *Language:* Python 3.x
- *Data Engineering:* Pandas, NumPy
- *Storage Architecture:* CSV (Flat-file Persistence)

## 🗂️ Dataset Architecture
The database is pre-loaded with a professional product catalog dataset containing logic-driven features: product_name, category, price, quantity, and discount.
